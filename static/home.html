Anti-Boredom

This application allows you to find an activity (thanks to the boredAPI) and, just for fun, gives you a joke (thanks to the jokeAPI) which contains words from the activity found.

Get activities and associated jokes

If you want to access the API and get activity ideas and related joke, go to:

https://marbotteantiboredom.herokuapp.com/ActJoke/

Replace by one of the following activity types:

  • education
  • recreational
  • social
  • diy
  • charity
  • cooking
  • relaxation
  • music
  • busywork

For example try : https://marbotteantiboredom.herokuapp.com/ActJoke/cooking

Export all the activity and jokes previously found

If you want to export all the previously found activities and jokes, go to

https://marbotteantiboredom.herokuapp.com/Export

The application will export a csv file
